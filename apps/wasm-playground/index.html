Atlas — In-Browser Verifier

Verify an Atlas trace bundle entirely in your browser. No network call. The same Rust verifier core that runs in the native CLI, compiled to WebAssembly. V1.14 — includes witness_failures wire-surface (Scope J).

A *.trace.json emitted by atlas-signer.

The *.pubkey-bundle.json the trace was signed against.

Auditor tooling switches on witness_failures[].reason_code for classification.


    


    

      Local dev: run
      wasm-pack build crates/atlas-verify-wasm --target web --release --out-dir ../../apps/wasm-playground/pkg
      from the repo root, then serve this directory with any static file server
      (npx serve apps/wasm-playground or
      python3 -m http.server). 

      Production: after V1.14 release, the published artefact
      is available at @atlas-trust/verify-wasm on npm.
