Atlas — In-Browser Verifier

Verify an Atlas trace bundle entirely in your browser. No network call. The same Rust verifier core that runs in the native CLI, compiled to WebAssembly. V1.14 — includes witness_failures wire-surface (Scope J). V1.16 — page hardened with strict CSP, Subresource Integrity on application JS, and Trusted Types enforcement.

A *.trace.json emitted by atlas-signer.
